* VCNL4000 Proximity and ambient light sensor.

Required properties:

  - compatible : should be one of
	"vishay,vcnl4000"
	"vishay,vcnl4010"
	"vishay,vcnl4020"
	"vishay,vcnl4200"
  - reg : the I2C address of the sensor
	0x13 for vcnl40x0
	0x51 for vcnl4200

Example:

&i2c1 {
	vcnl4010@13 {
		compatible = "vishay,vcnl4010";
		reg = <0x13>;
	};
};
